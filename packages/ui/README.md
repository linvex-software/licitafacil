# @licitafacil/ui

Biblioteca de componentes UI compartilhados para o monorepo Licitafacil.

## Status

🚧 **Em Desenvolvimento** - Componentes serão adicionados conforme necessário.

## Uso

```typescript
import { Button, Card } from "@licitafacil/ui";
```

## Estrutura Planejada

```
src/
├── components/

│   ├── Card/
│   ├── Input/
│   ├── Modal/
│   └── ...
├── hooks/
│   └── ...
└── utils/
    └── ...
```

## Desenvolvimento

```bash
# Build
pnpm build

# Watch mode
pnpm dev

# Type checking
pnpm typecheck

# Lint
pnpm lint
```
